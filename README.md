Sudoku-Solver
=============

A Sudoku Solver implemented in C++
by ioeric
=============

Usage: 
    make
    ./solver < testfile
    (To check solution) 
    make validator
    ./solver < testfile | ./validator

Input:  (stdin)
        81 characters in '0'-'9' or '.'(which means a uncertain cell)
